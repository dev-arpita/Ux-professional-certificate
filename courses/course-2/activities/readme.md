### Screenshorts 
When empathizing with users for a new design project, you should use:

- User Survey / interviews - 
    - representative sample
    [Survey data collection sheet](/courses/course-2/activities/initial-survey.png)

- Empathy maps- 
    - empathy maps to break down research data, crucial tool for processing the results of your interviews 
        [Empathy map](/courses/course-2/activities/empathy-map.png)

- User personas
    - Personas are fictional users who represent the needs, goals, and characteristics of larger groups.
        [Persona 1](/courses/course-2/activities/persona-01.png)
        [Persona 2](/courses/course-2/activities/persona-02.png)

- User stories
    A user story is a key part of a user persona. Told from a persona’s point of view, it introduces the user, lays out an obstacle, and states the user’s ultimate goal and reasons for that goal. Designers and design teams use these stories to identify and prioritize the key user goals and frustrations that will shape their designs.
        [User story 1](/courses/course-2/activities/user-story-01.png)
        [User story 2](/courses/course-2/activities/user-story-02.png)

- User journey maps :
     a user journey map is the series of experiences a user has as they achieve a specific goal. A user journey map is used to help UX designers understand the challenges a user may face when trying to use a product.
    [User journey map](/courses/course-2/activities/user-journey-map.png)
